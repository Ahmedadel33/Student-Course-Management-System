Student Course Management System

A **C++ console application** to manage students and their enrolled courses with a simple menu-driven interface. The system demonstrates **OOP principles, STL containers, and algorithms**.

___ Features
- Add, remove, and search students
- Display all students
- Enroll students in courses
- Show student courses
- Sort students by GPA

___ Project Structure
| File                     | Purpose                          |
|---------------------------|----------------------------------|
| `Person.h / Person.cpp`    | Base class with `name`          |
| `Student.h / Student.cpp`   | Inherits from `Person`, stores `id`, `GPA`, courses |
| `main.cpp`                 | Menu logic and program entry point |


Technologies
- C++ (OOP, STL)
